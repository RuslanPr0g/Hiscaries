# Start the app

Run the appropriate startup script based on what the user needs:

- **Full stack:** `./scripts/start.sh`
- **Backend only:** `./scripts/start-backend.sh`
- **Frontend only:** `./scripts/start-frontend.sh`
- **LAN/host mode:** `./scripts/host.sh`

Run the script from the repo root (`/home/thor/Desktop/projects/Hiscaries`).
After starting, report which services launched and any errors from the output.
